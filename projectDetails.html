Project Details 📂

Alex's Cookie Company🍪 :

"Alex's Cookie Company" was one of the first projects I created to learn and test basic HTML. It’s a simple, static website featuring:

- A home page with a custom logo and introductory paragraph.

- A recipe page that uses lists to display ingredients and step-by-step instructions for making cookies.

- An order form page including a menu, where users can submit cookie orders.

This project was a foundational exercise in web development, teaching me how to structure content, create links between pages, and build simple forms using only HTML. By comparing this early project to my current GitHub profile page—built with a combination of HTML and CSS—you can see the progression of my skills:

- Enhanced design and layout using CSS for styling and responsive design.

- More polished, professional aesthetics with modern web techniques.

- A deeper understanding of structuring content for clarity and usability.

"Alex's Cookie Company" represents my starting point in web development, while my GitHub profile reflects my growth and ability to create more sophisticated, visually appealing, and user-friendly projects.
